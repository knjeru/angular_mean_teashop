# The MEAN Little Tea Store

Visit this fine online tea establishment, and order a few of the rarest digital tea bags this side of the "Amazon"!

### Technologies Used:
* Mongo/Mongoose
* Express
* AngularJS
* Node.js
* SASS (Bourbon-neat)
* JSON Web Tokens (Authorization)
